During my professional tenure at the various architecture firms I worked at, I noticed a significant difficulty in the design of stairs by the architectural team. Due to its complex modeling, most current architectural software demonstrates an inefficient workflow for this. With this in mind, I decided to create a Grasshopper script to facilitate this process, aiming to develop a method that could be replicated by anyone, even without any knowledge of this language.
Durante o meu período de atuação profissinal nos diferentes escritórios de arquitetura pelos quais passei, percebi uma grande dificuldade na concepção de escadas por parte da equipe de arquitetos. Por se tratar de uma modelagem complexa, grande parte dos softwares de arquitetura atuais demonstram ter um workflow pouco eficiente para isto. Com isso em mente, resolvi criar uma programação em Grasshopper para facilitar esse processo com o objetivo de elaborar um método que poderia ser reproduzido por qualquer um, mesmo sem ter qualquer conhecimento dessa linguagem.
With my script in hand, the first step would be to draw your staircase plan in Rhino and organize the layers according to the structure described in Grasshopper. This plan should contain the perimeter of the staircase, the tread lines, the perimeter of the landing, and a guide curve that passes through the middle of each step, as seen in the image below. Additionally, it is necessary to create the staircase plan for each floor, as well as an extra one for the roof floor, with each plan at its correct level. This is because Grasshopper calculates the floor height based on the distance between the staircase plans. It is important to use the format described in the script, as I used the Dynamic Geometry Pipeline command from the Human plugin to import the geometries from Rhino to Grasshopper according to their layers to automate this process.
Tendo o meu script em mãos, o primeiro passo seria desenhar no Rhino a planta de sua escada e organizar os layers de acordo com a estrutura descrita no Grasshopper. Essa planta deve conter o perímetro da escada, as linhas de pisada, o perímetro do patamar e uma curva guia que passa pelo meio de cada degrau, como visto na imagem abaixo. Além disso, é necessário criar a planta da escada de cada pavimento além de uma extra para o pavimento de cobertura, sendo que cada planta deve estar no seu nível correto. Isso se dá pelo fato do Grasshopper calcular o valor do entrepiso baseado na distância entre as plantas da escada. É importante utilizar formato descrito no script, pois utilizei o comando Dynamic Geometry Pipeline do plugin Human para importar as geometrias do Rhino para o Grasshopper de acordo com os seus layers para automatizar esse processo.
This entire script was developed using clusters to better control the organization of the file, thus enabling the selection of specific outputs for your project (more details on this later). The second step would be to connect the "LAYERS" cluster to the "SUBD GUIDE CURVE" cluster according to their respective input and output names to generate guide curves for creating a SubD that will act as a ceiling for your staircase. I believe that SubD is a very interesting option for modeling the bottom of the staircase, as it allows for great freedom.
Essa programação inteira foi desenvolvida usando clusters para melhor controlar a organização do arquivo, possibilitando, pois, escolher outputs específicos para o seu projeto (mais detalhes sobre isso mais para frente). O segundo passo seria conectar o cluster "LAYERS" ao cluster "SUBD GUIDE CURVE" segundo os seus respectivos nomes de entrada e saída para gerar curvas guia para a criação de um SubD que atuará como um forro para a sua escada. Acredito que o subd seja uma opção muito interessante para a modelagem do fundo da escada, pois ela permite uma liberdade muito grande.
Now, it is necessary to "bake" the guide curves and use the "SubDLoft" command in Rhino to create a base SubD, which can be modified as needed. Do not forget to add this SubD to the SubD command of the script interface in Grasshopper.
É preciso agora dar "bake" nas curvas guia e utilizar o comando "SubDLoft" no Rhino para criar uma SubD base, que pode ser alterada conforme as suas necessidades. Não se esqueça de adicionar essa SubD ao comando de SubD da interface da programação no Grasshopper.
Once this is done, the script will model your staircase based on the provided information. It is important to note that this model will automatically update as the plan and the ceiling SubD are modified.
The "STAIRS - FULL" command creates a staircase with a full base, not considering the SubD created in the previous steps.
Com isso feito, o script modelará a sua escada com base nas informações fornecidas. É importante frisar que esse modelo atualizará automaticamente conforme a planta e o SubD de forro forem modificados.
O comando "STAIRS - FULL" cira uma escada com uma base cheia, não levando em consideração o SubD criado nos passos anteriores.
The "BOOLEAN SUBD" command interrupts the previous model with the SubD, which can be added to the "STAIRS - FULL" command and/or the "LANDING" command.
O comando "BOOLEAN SUBD" interrompe o modelo anterior com o SubD, podendo ele ser adicionado ao comando "STAIRS - FULL" e/ou ao comando "LANDING".
